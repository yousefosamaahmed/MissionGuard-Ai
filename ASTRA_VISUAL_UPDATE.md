# Astra-inspired visual presentation

This build keeps the MissionGuard AI models, data, PostgreSQL integration,
page navigation, upload workflow, reports, tests, and mission-analysis logic
unchanged. Only the presentation layer in `app.py` was updated.

The interface now uses a monochrome cosmic visual system inspired by the
referenced Astra Framer template: split black/white hero composition, large
typography, star-field details, rounded editorial cards, pill controls,
responsive spacing, and subtle motion.

No proprietary Astra images, text, or template assets are included.
