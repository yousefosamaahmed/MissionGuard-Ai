# Real OPSSAT-AD Data

This folder contains the real OPSSAT-AD v2 data supplied by the user:

- `raw/dataset.csv`: official engineered segment features.
- `raw/segments.csv`: original telemetry samples.
- `processed/official_test_predictions.csv`: packaged model predictions on the official test split.
- `upload_samples/`: real held-out CSV examples for application testing.
